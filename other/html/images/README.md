# 图片资源目录

请将以下 IDA Pro 反编译截图放置在此目录中：

## 需要的图片文件

### 控制流图对比
- `before-cfg.png` - 加固前的控制流图（建议 800x600px）
- `after-cfg.png` - 加固后的控制流图（建议 800x600px）

### 反编译代码对比
- `before-decompile.png` - 加固前的反编译伪代码（建议 800x600px）
- `after-decompile.png` - 加固后的反编译伪代码（建议 800x600px）

### 字符串对比
- `before-strings.png` - 加固前的字符串列表（建议 800x600px）
- `after-strings.png` - 加固后的字符串列表（建议 800x600px）

### VMP 架构图
- `vmp-architecture.png` - VMP 保护原理架构图（建议 1000x600px）

## 截图建议

1. **控制流图**：在 IDA Pro 中打开函数，切换到 Graph View，截图整个控制流图
2. **反编译代码**：在 IDA Pro 中切换到 Pseudocode View，截图伪代码
3. **字符串列表**：在 IDA Pro 中打开 Strings 窗口，截图字符串列表
4. **架构图**：可以使用流程图工具（如 draw.io）绘制 VMP 执行流程

## 图片要求

- 格式：PNG（推荐）或 JPG
- 分辨率：建议 800x600px 或更高
- 文件大小：建议控制在 500KB 以内
- 清晰度：确保文字清晰可读
